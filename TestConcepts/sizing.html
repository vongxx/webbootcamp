Sizing

  1. Content affect size(width of inline elements, height of all elements)
  2. Layout order is dependent on the html code sequence
  3. Child elements lies on top of parent (z-axis)

Position property

  1. Static (default)
    • Keep everything as default. According to HTML rules
  2. Relative
    • Relative to where it should have been in Static
    • To use with property top/left/right/bottom = ?px (adding a margin)
    • Will leave an empty space. no other element can take the empty space.
  3. Absolute
    • Relative to its parent
    • To use with property top/left/right/bottom = ?px (adding a margin)
    • Will not leave an empty space. Other html element will take its place.
  4. Fixed
    • Relative to viewport
    • Will be fixed to that position even if you scroll
    • Can implement a fixed nav bar
