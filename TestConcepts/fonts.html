font-family

  1. Font declaration
    • font-family: verdana, sans-serif
  2. Solving rendering errors: Users may not have the font you intended!
    • Use CSS Web Safe fonts - common fonts which a very high percentage of ppl will have
    • Or you can embed fonts eg. Google Fonts

font-size

  1. Fixed
    • px - Will not change if you change the browser setting's font size
    • Will change if you use the zoom in/out
  2. Relative
    • em, rem, %
    • 1 em = 1 rem = 100% = 16px
    • em is inherited from parent's font size, if parent is already 2em , then if you set 2 em in a child element, the final size will be 4 em !
    • rem is inherited from the base HTML document body font size (default = 16px) (Recommended!)
